## Setup
- Add `nuxt-module-auth0` dependency using yarn or npm to your project
- Add `nuxt-module-auth0` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'nuxt-module-auth0',

    // With options
    ['nuxt-module-auth0', { /* module options */ }],
 ]
}
```
